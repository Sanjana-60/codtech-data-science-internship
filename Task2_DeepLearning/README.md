# Task 2 – Deep Learning Project

## Project Title

Handwritten Digit Classification using TensorFlow

## Objective

The goal of this project is to build a deep learning model for image classification using the MNIST dataset.

## Tools and Technologies

* Python
* TensorFlow
* NumPy
* Matplotlib

## Dataset

MNIST Handwritten Digit Dataset

* 60,000 training images
* 10,000 testing images
* Each image size: 28 × 28 pixels

## Steps Performed

1. Loaded MNIST dataset using TensorFlow
2. Normalized image pixel values
3. Built a neural network model using Dense layers
4. Trained the model for 5 epochs
5. Evaluated model performance on test data
6. Visualized training accuracy

## Result

The trained deep learning model achieved approximately **97% accuracy** on the test dataset.
